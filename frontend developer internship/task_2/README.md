# Frontend Developer Challenge Quiz App

## Project Overview

Frontend Developer Challenge is an interactive quiz application built using HTML5, CSS3, and JavaScript. The application allows users to test their frontend development knowledge through multiple-choice questions. It provides live score tracking, progress updates, rank generation, accuracy calculation, and a leaderboard system for an engaging user experience.

## Features

* Welcome Screen with Name Input
* Interactive Multiple Choice Questions
* Live Score Tracking
* Progress Bar
* Rank Generation System
* Accuracy Calculation
* Certificate Style Result Screen
* Leaderboard using Local Storage
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage
